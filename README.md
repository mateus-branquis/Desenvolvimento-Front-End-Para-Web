# ONG de Exemplo

[Repositório do projeto](https://github.com/mateus-branquis/Desenvolvimento-Front-End-Para-Web/)

## Introdução

Este projeto tem como objetivo o desenvolvimento de uma plataforma web voltada para Organizações Não Governamentais (ONGs), oferecendo uma presença digital profissional, funcional e acessível. A proposta aplica fundamentos de HTML5, garantindo semântica, acessibilidade e responsividade.

Nesta primeira etapa do desenvolvimento, o foco foi na criação de páginas estáticas que representam as principais áreas da plataforma, apresentando informações institucionais, projetos sociais e um sistema de cadastro de voluntários e apoiadores.

## Desenvolvimento

A plataforma foi estruturada em três páginas principais: `index.html`, `projetos.html` e `cadastro.html`, atendendo aos requisitos propostos.

- **index.html**: informações sobre a organização, missão, visão, valores e formas de contato.  
- **projetos.html**: lista de projetos sociais, oportunidades de voluntariado e formas de doação.  
- **cadastro.html**: formulário interativo e validado em HTML5 para cadastro de voluntários, doadores ou interessados.

A estrutura de pastas segue boas práticas, com diretórios separados para CSS, JavaScript e imagens (assets), facilitando manutenção e escalabilidade.

Foram aplicados conceitos de semântica HTML5, utilizando corretamente `header`, `nav`, `main`, `section`, `article` e `footer`. Todas as páginas foram validadas no [W3C Validator](https://validator.w3.org/), assegurando conformidade com os padrões da web.

O projeto também é responsivo, oferecendo boa experiência em desktop, tablet e mobile. Imagens foram otimizadas para melhor desempenho e menor tempo de carregamento.
